Buttons

Help      Menus

POWER
    The power button; turns the machine on and off.

Floppy 1 and Floppy 2
    Use these to insert 5¼" "floppy disks" to / from the two floppy disk drives.
    These disks exist as 140K (143,360 byte) files, usually with wthe suffix ".dsk".
    Note that such disk images cannot be used in the "HD" drives below them.
    If the an inserted disk is a ProDos8 disk, it's volume name is displayed just
    below the button. (This volume name, which is stored in the disk, is NOT the name
    of the image file.)
    If the disk is an Apple Dos3 disk, "DOS3" and the "volume number" of the disk is displayed
    instead. For various reasons which we can't go into here, this number is usually 254.
    (In BASIC, use PR#6 or IN#6 to boot from 'Floppy 1'.)

HD 1 and HD 2
    Use these to insert or eject 3½" floppy (800K) or hard-disk (32M) images.
    Just as 5¼" disk images can't be used here, these can't be used in the drives above.
    (In BASIC, use PR#5 or IN#5 to boot from 'HD 1'.)

Tape
    *** This button is disabled in v0.1****
    Use this to insert or eject a cassette tape image. The data on the "tape" may be a BASIC
    program, text, or any binary data.

RESET
    Use this to interrupt a running program and reset the BASIC interpreter, just as on a
    physical Apple II.

Big Screen (or Small Screen)
    This doubles or halves the size of the screen.
